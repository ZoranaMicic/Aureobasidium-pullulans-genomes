# Aureobasidium-pullulans-genomes
# Genome assemblies of Aureobasidium pullulans (TaxID: 5580)
## Project goal
To download, inspect and analyze publicly available genome assemblies of Aureobasidium pullulans from NCBI.

## Data source
NCBI GeneBank assemblies retrieved using:
- TaxID: 5580
- NCBI FTP assembly summary

## Workflow overview
1. Download assembly metadata from NCBI
2. Filter assemblies by taxid
3. Download genome FASTA files
4. Perform quality checks and downstream analyses

All commands and scripts used to retrieve and analyze the data are documented in the 'scripts' directory.
